# OmniStack11_FrontEnd
Frontend Web feito em react.js

Criação deste projeto 

    - npx create-react-app frontEnd
    - npm install react-router-dom
    - npm install react-icons
    - npm install axios

Estrutura de pastas

    - criar pasta pages
         - criar pasta Logon
            - index e styles
         - NewIncident
            - index e styles
         - Profile
            - index e styles
         - Register
            - index e styles

    - na raiz do src:
        - App.js
        - routes.js
        - index.js
        - global.css
